[ ] pymacos Compilation for Win

    [ ] Require: - Intel Fortran IFX Compiler
                 - Visual Studio 2022
                 - Python >= 3.13.x  with at latest: Numpy, scipy, matplotlib

    [ ] clone github repo MACOS_API

    [ ] Anaconda, Standalone Python, UV Package Manager, ...
        [ ] MACOS/SMACOS
            [ ] open a cmd window
            [ ] activate Python 3.13 and initialise Intel Compiler with MS Visual
                e.g.:

                  [ ] activate py313            (Anaconda: if Python Env. is "py313")
                  [ ] .venv\Scripts\activate    (Python Virtual Environment)

                  [ ] Intel:
                     C:\"Program Files (x86)"\Intel\oneAPI\setvars intel64 vs2022


            [ ] Compile MACOS to create ".\libs" & ".\mod"
                cd [PATH TO MACOS_API Git Repo]\MACOS_API\pymacos\src\macos\cmake
                mkdir build
                cd build
                cmake -G "NMake Makefiles" -S ..
                nmake

                => this will create folders: ".\libs" & ".\mod"

        [ ] compile pymacos
            [ ] open a cmd window
            [ ] activate Python 3.13 and initialise Intel Compiler

            [ ] compile to create Python DLL
                cd [PATH TO MACOS_API Git Repo]\MACOS_API\pymacos\src\cmake   <<< need to update path
                mkdir build
                cd build
                cmake -G "NMake Makefiles" -S ..
                nmake

                => this will place the DLL (pymacosf90.cp313-win_amd64.pyd) into the pymacos folder
                => ensure that the DLL is not being used in an open Python session
